# Dragon Loot

## Description

Adds the ability to add custom loot to drop when killing the ender dragon.

## Options

| Name    | Description                    | Type      | Default |
| ------- | ------------------------------ | --------- | ------- |
| enabled | Whether this tweak is enabled. | boolean   | false   |
| drops   | A list of items to drop.       | item list | ELYTRA  |
